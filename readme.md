## Install

```
$ npm install -g bible-male-names-cli
```

## CLI

```
$ bible-male-names-cli --help
	
Usage
  $ bible-male-names <options>

Examples
  $ bible-male-names
  Abraham

  $ bible-male-names --all
  Aaron
  Abagtha
  Abda
  ...

Options
  --all  Get a list of all male names from The Bible
```
